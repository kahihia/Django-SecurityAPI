Files and Folders' Organisation
===============================

After going to the new :code:`Django-SecurityAPI` folder you'll will have different types of files and folders.

Here is the organisation of the project you're going to find :

+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|**Files & Folders**                         |                                                   **Description**                                         |
+============================================+===========================================================================================================+
|:code:`/application `                               |     The folder that contains :code:`settings.py` and :code:`local settings.py`                                        |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`/api`                                        |      The main folder : defines the code base of this app.                                                 |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`/functions`                                  |      A folder that contains some functions that can be used later                                         |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`/registration`                               |A folder that contains some templates for the login page (optionnal)                                       |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`manage.py`                                   |   The main file Django uses to run the server, to manage database.                                        |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`requirements.txt `                           |                     The packages our App needs to work                                                    |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`runtime.txt`                                 |                     The version of Python to use on for Heroku                                            |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`app.json `                                   |    "[a manifest format for describing web apps](https://devcenter.heroku.com/articles/app-json-schema)"   |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`*.bat`                                       |       Some Windows scripts used as alias for some django commands                                         |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`.env.dist`                                   |  A template of the file that describe some environnement variables                                        |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
|:code:`Procfile & Procfile.windows `                |    If you user Heroku only : describe actions the dynos have to do (optionnal)                            |
+--------------------------------------------+-----------------------------------------------------------------------------------------------------------+
